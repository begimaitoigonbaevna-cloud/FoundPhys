# Добро пожаловать на FoundPhys

FoundPhys — образовательный проект, посвящённый основам физики. Наша цель — сделать физику понятной, простой и увлекательной для учеников и всех, кто интересуется наукой.

На сайте вы найдёте:
- Понятные объяснения тем, от классической механики до современной физики.
- Примеры из реальной жизни для наглядного понимания законов физики.
- Формулы, иллюстрации и простые упражнения для самостоятельного изучения.
- Интерактивные примеры для визуализации физических явлений.
